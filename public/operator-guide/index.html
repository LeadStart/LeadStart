If you only have 5 minutes

The crucial bits
Read these three things before touching anything

1. The production URL is leadstart-ebon.vercel.app. Anything you do there is live for paying clients.
2. The Instantly webhook is a one‑time registration on /admin/settings/api. Once flipped on, every reply hits our pipeline. Don't flip it until a test client is set up.
3. Pause / resume / delete on a campaign updates state immediately — there is no "undo" prompt.
